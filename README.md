# COCET FLASH CARD

フラッシュカードどタイピング同時にできたら効率的じゃね？
という謎の理屈により誕生しました。

## 使い方:
見りゃわかる

## 注意事項:
スマホには対応していません。
そしてLinuxとmacOSは動作テストをしていません。
Windowsしか勝たん。

## コードの中身を覗きたい人へ

[Svelte](https://svelte.jp/)と[Tauri](https://tauri.app/)で開発されています。

staticにコード以外のファイルが、srcにコードが、そしてsrc-tauriにTauri関連のファイルがあります。

Svelteって楽しいね(=
